# SIT305
A collection of my work for SIT305 - Mobile Application Development.   

## Folder Structure
- **2.1P** contains the work for task 2.1P, that being a basic unit converter app.
- **QuizTime** contains the work for task 3.1C, it is a basic quiz app that introduced me to multi-activity apps. ***I would like to come back to this and evolve it***.
- **Personal Planner** is the work for 4.1P, it allows users to save events via the Room Persistence Library and features multiple screens as fragments rather than activities.
- AdditionalApps contains various apps developed over the course of the trimester that are not a part of any task.
- **5.1C** is actually 2 applications. The first, **SportsNews** is a news app that experiments with multiple recycler views and fragment management. The second, **iStream** is an where users log in and view Youtube videos that they can save to their playlist. This one also uses fragments and recycler views as well as Room persistence and YouTube's API in a WebView.
- **6.1D** is a quiz app expanding QuizTime with various features that we have learned since, including AI! (I want to throw up)
- **7.1P** is a Lost and Found app that uses SQLite instead of Room persistence to store data on lost and found objects.
-  **8.1C** is an AI chatbot.
- **9.1P** is an extension of the Lost and Found app that enables Google Maps support.
- **10.1D** is an extension of 6.1D that adds more history via SQLite, enables payments via Google pay and sharing via intents.
