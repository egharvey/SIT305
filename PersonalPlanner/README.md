# Personal Planner 4.1P
This project is a Personal Planner made for SIT305 Task 4.1P using both Fragments and Room data persistence. In one fragment, you can save new events and in another you can view all events in a recycler view. Clicking on an item will bring up more details, where the event can be updated or deleted.

## Important Folders  
- *app/src/main/java/com/example/personalplanner* features java files, including those running each activity or fragment as well as those defining events, the room database and the dao that interacts between them.
- */app/src/main/res/layout* features files that define the layout of each fragment or activity as well as the layout that is used for each item in the recycler view.
- */app/src/main/res/menu* has the layout design for the bottom navigation menu.
- */app/src/main/res/drawable* features various icons used in the app.
