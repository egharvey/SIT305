# Talk to a Wall
A basic chatbot app using Ollama.

## Folder Structure
- app/src/main/java/com/example/talktoawall contains all the java files used in the app including activities, classes, database files, etc.
- app/src/main/res/layout contains layout files for activities and elements for recycler views
- app/src/main/res/drawable contains image files used throughout
- app/src/main/res/values contains definitions for stuff such as colours and strings
