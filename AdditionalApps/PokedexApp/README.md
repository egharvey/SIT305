# Pokdedex React Native
This is a basic React Native application that utilises [Pokepi](https://pokeapi.co/) to run a basic pokdex.

### Installation and Setup
First run <code>npm i</code> to install dependencies.
Next run <code>npx expo start</code> to run expo. Using expo you can connect via your browser, a mobile emulator or your phone to run the app. (Note that this app has not been tested in browser).
Lastly, while working with the app use <code>npm run lint</code> and <code>npm run lint -- --fix</code> to check for errors and formatting issues to maintain consistency and quality in code.
